# ScamBlocker

AI-powered scam call protection for UK landlines and mobiles.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Connected to Vercel for automatic deployments.

## Environment Variables

For production, set these in Vercel:

- `VITE_SUPABASE_URL` - Supabase project URL
- `VITE_SUPABASE_PUBLISHABLE_KEY` - Supabase anon key
